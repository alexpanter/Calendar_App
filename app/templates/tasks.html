Tasks mode

Tasks mode allows you to browse all coming tasks in an orderly fashion, and rearrange them as you see fit.

Tasks can be sorted into categories, and you can change their significance values, making them appear in a sorted manner.

New tasks can be added, and when a Task is done, marking it as done will remove it from the list. (it will however show up in your history)
